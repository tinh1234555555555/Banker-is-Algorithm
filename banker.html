Input
Instruction
  1. Select "Input" to enter the number of processes and resources.
  2. Select "Generate Input Fields" to initialize the "Allocation", "Max" and "Available" tables.
  3. Then, enter the desired metrics.
  4. Select "Generate Need Matrix" to calculate the "Need" matrix.
  5. Continuously click "Start Process" to observe how the algorithm operates.
  6. Select "Output" to display the results.
  7. Select "Reset" to delete everything.
  8. Select "Edit" to edit the data to suit your new purpose. Then repeat as instructed.
  9. Select "Request" to display the "Request" matrix.
    • Select "Enforce" to check the data. Then repeat as instructed.
